# Cargo Net

Sitio web oficial de Cargo Net — courier internacional B2B para PYMEs argentinas.

## Páginas

- `/` — Home
- `/como-funciona` — Proceso de importación paso a paso
- `/cotizar` — Cotizador de envíos
- `/aduana-y-requisitos` — Requisitos aduaneros
- `/seguimiento` — Seguimiento de envíos
- `/sobre-nosotros` — Empresa
- `/contacto` — Formulario de contacto
- `/blog` — Contenido editorial

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- TypeScript
- Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Deploy

El proyecto se despliega automáticamente en Vercel al hacer push a `main`.
